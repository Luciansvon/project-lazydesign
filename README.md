# Financial & Personal Assistant Bot

Bot analisis keuangan dan asisten pribadi lengkap yang mendukung kegiatan sehari-hari.

## Fitur Utama

### 📊 Analisis Keuangan
- Tracking pemasukan dan pengeluaran
- Kategorisasi transaksi otomatis
- Visualisasi budget dan spending
- Laporan harian, mingguan, bulanan
- Prediksi keuangan berbasis AI
- Integrasi dengan bank (via API)

### 🤖 Asisten Pribadi
- Manajemen tugas dan to-do list
- Pengingat jadwal dan deadline
- Catatan harian (journaling)
- Analisis produktivitas
- Rekomendasi berdasarkan kebiasaan

### 💡 Teknologi
- Python 3.9+
- Telegram Bot API / Discord
- SQLite / PostgreSQL untuk database
- Pandas untuk analisis data
- Scikit-learn untuk prediksi
- Plotly untuk visualisasi
- OpenAI API / Local LLM (Ollama)

## Instalasi

```bash
pip install -r requirements.txt
```

## Konfigurasi

1. Copy `.env.example` ke `.env`
2. Isi token bot dan API keys
3. Jalankan bot: `python main.py`
